# Project Plan to Save Southport Pier

## Current Status
The pier is currently closed due to structural issues identified in recent surveys.

## Objectives
1. Raise necessary funds for repairs
2. Complete structural assessment
3. Undertake essential repairs
4. Modernize facilities
5. Reopen to the public

## Timeline
- Phase 1: Assessment and Planning (3 months)
- Phase 2: Fundraising Campaign (6-12 months)
- Phase 3: Repairs and Renovation (12-18 months)
- Phase 4: Reopening and Future Maintenance Planning
